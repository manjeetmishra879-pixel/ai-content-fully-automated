# Complete 60+ Engine Reference

## Content Generation Engines

1. **Trend Engine** - Fetches trends from Google, YouTube, Reddit, RSS, social signals
2. **Real-Time Viral Radar Engine** - Detects rising fast trends (X/15m, Google/30m, Instagram daily, TikTok/6h)
3. **Competitor Analysis Engine** - Analyzes top creators, viral content, audio trends, structure/pacing/visuals
4. **Trend Hijacking Engine** - Adapts trending formats, hashtags, audio; fallback to evergreen
5. **Category Engine** - Classifies content (motivation, business, education, entertainment, religion, tech, news)
6. **Category Router Engine** - Routes by category/platform (News→X, Motivation→Instagram, etc.)
7. **Content Engine** - Generates scripts, hooks, CTA via Ollama/LLaMA/Mistral
8. **Viral Hook Engine** - Generates curiosity/shock/value hooks, pattern interrupts
9. **Hook Library Engine** - Maintains viral hook templates and variations
10. **Translation Engine** - Translates to multiple languages (LibreTranslate, Helsinki-NLP)
11. **Caption Engine** - Generates captions, hashtags, SEO tags
12. **Comment CTA Engine** - Generates engagement CTAs (Agree? Part 2?, etc.)
13. **Voice Engine** - Text-to-speech narration (Coqui TTS, on-demand loading)
14. **Subtitle Engine** - SRT/VTT subtitles from script/Whisper
15. **Asset Fetch Engine** - Fetches stock images/videos from Pexels, Pixabay (with caching)

## Media Production Engines

16. **Image Generation Engine** - Quote cards, banners, posters, memes (Pillow, CPU-safe)
17. **Image Engine** - Advanced image variants for carousel, posters
18. **Video Engine** - FFmpeg-based reels, shorts, long-form with stitching, subtitles, audio sync, transitions
19. **Humanized Content Engine** - Talking head avatars, reaction clips, AI voice with emotion
20. **Thumbnail Engine** - Generates thumbnails with bold text, contrast, emotional triggers
21. **Expert Editing Techniques Engine** - Zoom cuts, speed ramps, glitch, fade, text animations
22. **Visual Enhancement Engine** - Auto sharpen, contrast optimize, rule of thirds, safe zones
23. **Audio Quality Engine** - Normalize -14 LUFS, music ducking, fade in/out

## Platform & Publishing Engines

24. **Platform Psychology Engine** - Platform-specific optimization (Instagram: aesthetic, TikTok: funny, etc.)
25. **Series Builder Engine** - Episode tracking, cliffhangers, part series management
26. **Content Bucket Engine** - Distribution mix (40% viral, 25% evergreen, 20% educational, 15% personal)
27. **Emotional Resonance Engine** - Targets inspiration, curiosity, urgency, humor
28. **Attention Optimizer Engine** - Hook in 3 sec, pattern reset/10 sec, mid-video cliffhanger
29. **Publisher Engine** - Multi-platform publishing (Facebook, Instagram, YouTube, Telegram, X, LinkedIn)
30. **Scheduler Engine** - Timezone-aware scheduling via Celery + Redis
31. **Category Router Engine** - Platform-specific routing

## Quality & Analytics Engines

32. **Quality Scoring Engine** - Publish >75, Review 60-75, Regenerate <60
33. **Engagement Prediction Engine** - Predicts watch time, skip rate, like/share probability
34. **A/B Testing Engine** - Tests hooks, thumbnails, captions, music, posting times
35. **Skip Analysis Engine** - Tracks 3s/10s/30s skips, learns and updates templates
36. **Best Time Learning Engine** - Learns optimal hour/day/timezone per platform
37. **Hashtag Learning Engine** - Tracks impressions/reach/engagement, boosts winners
38. **Content Freshness Engine** - Enforces 24h same-keyword, 4h same-category, 0.7 similarity limit
39. **Anti-Duplication Engine** - Uses embeddings, image hash, video fingerprint
40. **Content Decay/Recycle Engine** - 30-day repost, 90-day repurpose, 60-day evergreen recycle
41. **Analytics Engine** - Tracks views, likes, CTR, retention, conversions
42. **Skip Analysis Engine** - Pattern learning from viewer behavior
